## 👨‍💻 Обо мне

Я — Python Backend Developer с опытом разработки backend-сервисов, REST API, интеграций с внешними сервисами и production-поддержки.

Работаю с Python, FastAPI, Django/DRF, PostgreSQL, SQLAlchemy, Redis, Celery, Docker и асинхронными сервисами. Есть опыт проектирования API, работы с базами данных, фоновых задач, интеграций, логирования, тестирования и деплоя.

Отдельное направление моего опыта — разработка AI/LLM-функциональности для production-продукта. Я участвовал в создании backend/gateway-слоя для AI-driven decision-making продукта на Python/Litestar: интегрировал внешний LLM API, реализовывал guardrails, structured output через JSON Schema, постобработку ответов модели, fallback-логику, обработку timeout/network/provider-side ошибок и тестирование API-сценариев.

До перехода в разработку работал в сфере финансов и аналитики, где занимался автоматизацией бизнес-процессов, работой с данными, отчётностью и анализом финансовых показателей. Этот опыт помогает мне лучше понимать бизнес-логику, требования пользователей и прикладную ценность backend-решений.
